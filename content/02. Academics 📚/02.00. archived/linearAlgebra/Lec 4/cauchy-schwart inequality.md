# cauchy-schwart inequality
### definition
- for any 2 vectors $x, y \in R^n$: $|x^Ty| \leq ||x||_2.||y||_2$
- Generalize:
	- $|x^{T}y|\le||x||_{p}||y||_{q},\mathrm{where}\left.{\frac{1}{p}}+{\frac{1}{q}}=1\right.$ 
	- "=" holds when $\frac{|x^Ty|}{||x||_p.||y||_q} = 1 \iff x = \alpha y$
