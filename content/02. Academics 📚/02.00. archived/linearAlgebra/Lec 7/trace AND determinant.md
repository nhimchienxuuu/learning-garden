# title
### description
- sum the diagonal elements in a square matrix
- The trace is a linear function of the entries of the matrix.  
- $trace A = trace A^T$ for any square matrix A.  
- $trace AB = trace BA$ for any matrices $A ∈ R^{n×m}, B ∈ R^{m×n}$.
